# Signup Form Web Application

## Project Description
This project is a simple **Signup Form Web Application** developed using HTML, CSS, and JavaScript. The purpose of this project is to create a user-friendly interface where users can register by entering their basic details.

## Features
- User-friendly signup form
- Input fields for user details
- Basic form validation using JavaScript
- Responsive design using CSS

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
signup-form-project
│
├── index.html
├── style.css
└── script.js

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in any web browser
